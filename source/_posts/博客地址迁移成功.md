一大早起来就开始慢慢迁移

1. 首先把 CDN 部署先全部拆掉
2. 然后是创建子域名
3. 然后本地部署调试完毕之后上传 GitHub
4. 然后 Netlify 连接 GitHub 进行线上部署
5. 部署之后改一下域名
6. 然后把我自己的域名绑定上
7. 最后把 CDN 部署一下
8. 从上午八点大约忙到下午两点多
